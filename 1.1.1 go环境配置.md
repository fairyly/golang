

# 1.安装 Go 语言

```
1. 访问 https://go.dev/ ，点击 Download ，下载对应平台安装包，安装即可
2. 如果无法访问上述网址，可以改为访问 https://studygolang.com/dl 下载安装
3. 如果访问 github 速度比较慢，建议配置 go mod proxy，参考 https://goproxy.cn/ 里面的描述配置，下载第三方依赖包的速度可以大大加快
```
